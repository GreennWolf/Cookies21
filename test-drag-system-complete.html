🧪 Prueba Completa del Sistema de Drag

Funcionalidades a probar:

1. 🎯 Drag de Componentes Raíz (Amarillos)

  • Método A: Arrastrar directamente el componente
  • Método B: Usar el botón azul ⛶ (hover)
  • Resultado: Debe moverse libremente por el banner
  • Attach: Si se suelta sobre el contenedor azul, debe attacharse como hijo

2. 📦 Drag en Contenedores (Azules)

  • Handle pequeño: Usar el handle azul en la esquina
  • Botón azul: Debe activar el handle automáticamente
  • Límites: Los hijos se mantienen dentro del contenedor

3. 🔄 Sistema de Attach/Detach

  • Attach: Arrastrar componente raíz sobre contenedor
  • Detach: Arrastrar hijo fuera del contenedor
  • Visual: Feedback de drag over en contenedores
